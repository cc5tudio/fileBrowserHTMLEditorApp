FileBrowserEditorApp
==============

A Web Based File Browser with jsTree, Angularjs, Expressjs and TinyMCE HTML editor

### Install

* Download/clone the repo
* Run `npm install`
* Run `gulp` and navigate to `http://localhost:3000` to view the app
